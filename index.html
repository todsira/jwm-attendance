First-Time Setup
Connect your Google Sheets backend. You only need to do this once — the URL is saved in your browser.
Setup steps:
  1. Open the Code.gs file in Google Apps Script
  2. Paste your Google Sheet ID into the SHEET_ID variable
  3. Run setupSheets() once to create all tabs
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me · Who has access: Anyone
  6. Copy the Web App URL and paste it below
